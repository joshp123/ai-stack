## Built‑in Clawdbot tools

- `bash` — Run shell commands (supports background sessions).
- `process` — Manage background bash sessions.
- `grep` — Search file contents for patterns.
- `find` — Find files by glob pattern.
- `ls` — List directory contents.
- `whatsapp_login` — Generate a WhatsApp QR code and wait for linking.
- `clawdbot_browser` — Control Clawd’s browser.
- `clawdbot_canvas` — Present/eval/snapshot the Canvas.
- `clawdbot_nodes` — List/describe/notify/camera/screen on paired nodes.
- `clawdbot_cron` — Manage cron jobs and wake events.
